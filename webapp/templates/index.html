Integer Humanization

Convert numbers into readable formats:

  • intcomma: Add commas to large numbers
  • intword: Convert to word form (millions, billions)
  • apnumber: Spell out numbers 0-9
  • ordinal: Convert to ordinal form (1st, 2nd, 3rd)

Date & Time Humanization

Make dates and times more natural:

  • naturaltime: Relative time (3 hours ago)
  • naturaldelta: Time span descriptions
  • naturalday: Today, yesterday, tomorrow
  • naturaldate: Natural date formatting

About This Application

This web application is built on top of the excellent Python humanize library, which provides utilities for converting data into human-readable formats. The app uses:

  • FastAPI: Modern, fast web framework for building APIs
  • Jinja2: Powerful template engine for HTML rendering
  • Python Humanize: Core library with all humanization functions

The original humanize module remains clean and reusable. All business logic is kept in the library, while the web app provides a user-friendly interface for interaction.

Quick Examples:
  • AI Text → Natural, human-like writing (AI Humanizer)
  • 1234567 → "1,234,567" (intcomma)
  • 1000000 → "1.0 million" (intword)
  • 3 hours ago → naturaltime
  • 1048576 bytes → "1.0 MB" (naturalsize)
  • 0.5 → "1/2" (fractional)
