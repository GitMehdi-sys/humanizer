intcomma() - Add Commas to Numbers

Convert an integer to a string containing commas every three digits.

Examples: 1234567 → "1,234,567" | 1234.56 → "1,234.56"

intword() - Convert to Word Form

Convert a large integer to a friendly text representation.

Examples: 1000000 → "1.0 million" | 1234567890 → "1.2 billion"
Use Python format strings like %.1f or %.2f

apnumber() - Spell Out Numbers

For numbers 0-9, return the number spelled out. Otherwise, return the number.

Examples: 4 → "four" | 10 → "10"

ordinal() - Convert to Ordinal Form

Convert an integer to its ordinal as a string.

Examples: 1 → "1st" | 2 → "2nd" | 3 → "3rd" | 4 → "4th"

Tips:

  • intcomma() works with both integers and floats
  • intword() is great for displaying large numbers in reports and dashboards
  • apnumber() follows AP (Associated Press) style guidelines
  • ordinal() handles special cases like 11th, 12th, 13th correctly
